UI Bootstrap More Info

UI Bootstrap (more info) is a set of native AngularJS directives based on Bootstrap's markup and CSS. This page demonstrates several of those components adapted to the look and feel of the Design System. All of these items also exist in the Design System as "plain" Bootstrap/Angular, and those are recommended over these.

The angular-ui bootstrap components require the design_system_bootstrap_ui.css stylesheet to be included in your project.
See index.html code comments for details.

Due to limitations in the bootstrap-ui directives and templates, additional classes may be required on components for proper styling
using ng-class. Some variations of components may not be available and are noted in the examples below.
